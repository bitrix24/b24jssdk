`Browser`{lang="ts-type"} is a singleton (`new BrowserManager()`) exported from `@bitrix24/b24jssdk`. It reads `navigator.userAgent` once at import time. On non-browser environments (Node) every check is safe — the manager falls back to `'?'`.

```ts
import { Browser } from '@bitrix24/b24jssdk'

if (Browser.isMobile()) {
  // ...
}
if (Browser.isLocalStorageSupported()) {
  // ...
}
```

## Browser Detection

| Method | Returns |
|----|----|
| `isOpera()` | `true` if UA includes `opera`. |
| `isIE()` / `isIE6/7/8/9/10/11()` | Heuristics for Internet Explorer versions. |
| `isSafari()` | `safari && !chrome`. |
| `isFirefox()` / `isChrome()` | Self-explanatory. |
| `detectIEVersion()` | Numeric IE version (`-1` if not IE). |

## OS / Platform

| Method | Returns |
|----|----|
| `isMac()` / `isWin()` / `isLinux()` | Desktop OS. |
| `isAndroid()` | UA includes `android`. `detectAndroidVersion()` returns the parsed version. |
| `isIPad()` / `isIPhone()` / `isIOS()` | iOS device. `isIPad()` also covers Mac with touch (modern iPadOS). |

## Capabilities

| Method | Returns |
|----|----|
| `isMobile()` | iOS / Android / `mobile` / `touch` UA. |
| `isRetina()` | `window.devicePixelRatio >= 2`. |
| `isTouchDevice()` | `'ontouchstart' in window` or `navigator.maxTouchPoints > 0`. |
| `isDoctype(target?)` | `compatMode === 'CSS1Compat'` (or fallback). |
| `isLocalStorageSupported()` | `setItem/removeItem` round-trip succeeds. |
