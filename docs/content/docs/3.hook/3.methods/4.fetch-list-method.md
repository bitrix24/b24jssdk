```ts
fetchListMethod(
  method: string,
  params ? : any,
  idKey ? : string,
  customKeyForResult ? : string | null
```

Calls a REST API list method and returns a generator object.

| Parameter            | Type           | Description                          |
|----------------------|----------------|--------------------------------------|
| `method`             | `string`       | Request method.                      |
| `params`             | `any`          | Request parameters.                  |
