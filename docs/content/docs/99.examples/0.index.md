This section collects task-shaped examples of using `@bitrix24/b24jssdk`. Most recipes are single, copy-paste-friendly TypeScript files that live in [`skills/b24jssdk-recipes/examples/`](https://github.com/bitrix24/b24jssdk/tree/main/skills/b24jssdk-recipes/examples) and are type-checked in CI via `pnpm run skills:typecheck`.

## Cookbook

Curated entry points — each recipe ships environment variables, an expected output sketch, and a `Limitations` block.

## Repository

All runnable examples live in two places:

- [`skills/b24jssdk-recipes/examples/`](https://github.com/bitrix24/b24jssdk/tree/main/skills/b24jssdk-recipes/examples) — the 17 SDK-native recipes above, type-checked in CI.
- [`bitrix24/b24sdk-examples`](https://github.com/bitrix24/b24sdk-examples) — paired UI showcases (Nuxt, React) and the original Node/Hook starter scripts. Notable folders:
  - `js/02-nuxt-hook` — Nuxt + `B24Hook` quick start
  - `js/03-nuxt-frame` — Nuxt + `B24Frame` (helper, Pull client, dialogs)
  - `js/05-node-hook` — pure Node.js + `B24Hook` script
  - `js/04-react-frame` — React + `B24Frame`
