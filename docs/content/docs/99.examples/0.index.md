These recipes are an early preview. Review the per-recipe `Limitations` block before using one in production.
::

## Cookbook

Each recipe is a single, copy-paste-friendly file with environment variables, an expected output sketch, and a `Limitations` block that calls out the known sharp edges. Pick the one closest to what you're building and adapt — the goal is "obvious starting point", not "exhaustive cookbook".
