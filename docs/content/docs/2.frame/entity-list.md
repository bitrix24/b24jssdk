To display a list of entities we use the following components:

- [B24Card](https://bitrix24.github.io/b24ui/docs/components/card/) - Render the content within a card component comprising a header, body, and footer section.
- [B24Table](https://bitrix24.github.io/b24ui/docs/components/table/) - A responsive data table component.
- [B24Pagination](https://bitrix24.github.io/b24ui/docs/components/pagination/) - A navigation component with buttons or links for pagination.
- [B24Empty](https://bitrix24.github.io/b24ui/docs/components/empty/) - An empty state component.
- [B24Button](https://bitrix24.github.io/b24ui/docs/components/button/) - A button capable of linking or performing an action.
- [B24Skeleton](https://bitrix24.github.io/b24ui/docs/components/skeleton/) - A toggle control for switching between two states.
